# mattrick.org
Matthew McCune's personal website and blog.

## Dependencies
In order to build this website, you'll need Zola. You can view installation instructions for your platform 
[here](https://www.getzola.org/documentation/getting-started/installation/).

To build frontend assets, you'll also need Node and NPM. Make sure to run `npm install` before trying to build.

## Building
To build the site, simply run `npm run build`. It will build all assets and invoke Zola to build the site itself.

## Development
In development, run `npm run dev` to watch assets and run a Zola development server.
